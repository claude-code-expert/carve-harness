CLASSIFICATION RULES (validated design — implement exactly):
- not in manifest AND in artifacts  -> new-recommended
- cur === next                      -> unchanged
- cur === orig && next !== orig     -> carve-updated
- cur !== orig (and not above)      -> user-modified
- orig === '' (un-migrated v1)      -> user-modified (conservative) + hint to run `carve migrate`
