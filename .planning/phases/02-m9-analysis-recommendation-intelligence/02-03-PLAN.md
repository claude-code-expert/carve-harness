    - readPrefs(root): reads <root>/.claude/.carve-prefs.json; returns null when absent; returns null (not throw) on malformed JSON.
    - writePrefs(root, prefs): creates <root>/.claude/ if needed, writes pretty JSON; round-trips with readPrefs.
    - applyPrefs(design, prefs): returns a string[] of default-checked ids = (design.recommended minus prefs.deselected) union prefs.selected, intersected with design.available (never check an unavailable id). Pure, deterministic, no I/O.
    - applyPrefs(design, null) === design.recommended membership (no prefs → unchanged).
  </behavior>

    Define and export:
    - `export interface CarvePrefs { deselected: string[]; selected: string[]; updatedAt: string }`.
    - A const for the relative path `'.claude/.carve-prefs.json'`.
    - `readPrefs(root: string): CarvePrefs | null` — if file missing return null; parse with try/catch, on failure return null (do not throw — corrupt prefs must not break install). Validate the parsed object has array `deselected`/`selected` (narrow `unknown`); if not, return null.
    - `writePrefs(root: string, prefs: CarvePrefs): void` — mkdirSync(dirname, {recursive:true}); writeFileSync pretty JSON (2-space). Confined to .claude/ (D5).
    - `applyPrefs(design: HarnessDesign, prefs: CarvePrefs | null): string[]` — pure. If prefs null, return [...design.recommended]. Else build a Set from recommended, delete each id in prefs.deselected, add each id in prefs.selected, then filter to ids in design.available. Return array.

    - readPrefs on a malformed .carve-prefs.json → null (no throw).
    - applyPrefs: recommended ['a','b','c'] available ['a','b','c','d'], prefs { deselected:['b'], selected:['d'] } → set membership {a,c,d}.
    - applyPrefs with null prefs → equals recommended.
