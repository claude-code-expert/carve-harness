# handoff — 세션 인계

세션을 넘길 때 아래를 한 파일(`.handoff.md` 또는 STATE)에 적는다.

- ✅ **무엇이 됐나** (검증된 증거와 함께)
- ❌ **무엇을 시도했고 실패했나** (반복 방지)
- ⏳ **다음 할 일 + 아직 안 한 것**

`precompact-handoff` 훅과 짝을 이뤄 압축 직전 상태를 영속화한다.
