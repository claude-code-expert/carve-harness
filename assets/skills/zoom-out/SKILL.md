# zoom-out — 시스템 수준 시야

> mattpocock/skills(MIT)의 zoom-out 패턴에서 영감.

- 세부 구현 대신 **상호 연결된 모듈과 그것을 호출하는 함수**를 프로젝트 도메인 용어로 매핑한다.
- 사용 시점: 낯선 영역 파악, 컴포넌트가 전체 아키텍처에 어떻게 들어맞는지, 구현 전 조망.
- 출력: 모듈 지도 + 데이터 흐름 + 진입점. 코드 나열이 아니라 관계를 설명한다.
