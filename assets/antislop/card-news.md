# 카드뉴스 (Card News)

> 전역 규칙(SKILL.md §2~§4) 위에 더해지는 추가 제약. 전역 게이트를 먼저 통과시킨 뒤 이 파일의 게이트를 통과한다.

## 산출물 사양 (MUST)
- **캔버스**: 정사각 `1080×1080` 기본, 세로형은 `1080×1350`(4:5). 한 묶음(carousel) 안에서 비율을 섞지 않는다.
- **구성**: `표지(1) → 본문(N) → 마무리/CTA(1)`. 카드당 **핵심 메시지 1개만**.
- **모바일 가독성**: 1080px 폭 기준 본문 최소 `36px`, 핵심 문장 `48~72px`, 표지 타이틀 `72~120px`. 한 카드 본문은 **3~4줄 이내**.
- **줄바꿈**: 한글은 **어절(띄어쓰기) 단위**로 끊고, 줄 끝에 한 글자만 남기지 않는다. 의미 단위로 직접 `<br>` 제어.
- **번호**: 카드 묶음이면 우하단/상단에 `02 / 07` 형식 페이지 인디케이터. (장식 아닌 정보)

## 추가 금지 (MUST NOT)
- 카드마다 다른 배경색·다른 그라데이션으로 "예쁘게" 만들기 금지. 배경은 **흰색 또는 단일 무채색** 고정.
- 숫자·키워드에 글로우/네온/외곽선 효과 금지. 강조는 **굵기·크기·액센트 1색**으로만.
- 말풍선·반짝이·도형 스티커·이모지 떡칠 금지. 일러스트가 필요하면 `svg-image.md` 규칙을 따른 **단색 라인 아이콘 1개**까지만.
- 폰트 3종 이상 혼용 금지(제목용 1 + 본문용 1, 최대 2종). 자간을 의미 없이 벌리지 않는다.
- 카드 전체를 덮는 큰 따옴표("") 워터마크, 모서리 장식 금지.

## 추가 강제 (MUST)
- **시각적 일관성**: 모든 카드가 동일한 여백 그리드(예: 좌우 96px 안전 영역), 동일한 타이포 스케일, 동일한 액센트 위치를 공유한다.
- **대비**: 본문 텍스트 대비비 **4.5:1 이상**(흰 배경 위 `#111`, 회색 보조는 `#6B7280`까지). 작은 화면에서 회색 본문 금지.
- **표지**: 제목 + 한 줄 부제 + 출처/계정명. 표지에서 핵심을 다 말하지 말고 "왜 넘겨야 하는지"만 준다.
- **여백이 주인공**: 안전 영역 밖으로 텍스트가 닿지 않게 하고, 카드의 40% 이상은 비워 둔다.

## 자가 점검 (출력 전 추가 통과 필수)
- [ ] 카드 간 배경/그리드/타이포 스케일이 통일됐는가? (불통일이면 재작성)
- [ ] 한 카드에 메시지가 2개 이상 들어갔는가?
- [ ] 1080px 기준 본문이 36px 미만인가?
- [ ] 줄 끝에 외톨이 글자(예: "다", "요")가 떨어졌는가?
- [ ] 이모지·스티커·반짝임이 장식으로 쓰였는가?

## Good / Bad 예시

**BAD** — 카드마다 다른 보라 그라데이션, 이모지 불릿, 글로우 숫자
```css
.card { background: linear-gradient(135deg,#a855f7,#ec4899); }
.num  { text-shadow: 0 0 24px #ec4899; }          /* 글로우 금지 */
.li::before { content: "✨"; }                      /* 이모지 불릿 금지 */
```

**GOOD** — 무채색 고정 배경, 액센트 1색, 위계는 크기·굵기로
```css
/* 폰트: 헤드라인 임팩트 위해 Noto Sans KR 900, 본문은 동일 패밀리 400 */
.card { width:1080px; height:1080px; background:#FFFFFF; padding:96px; }
.card .kicker { font:700 28px 'JetBrains Mono'; color:#D97706; letter-spacing:.08em; }
.card .head   { font:900 88px 'Noto Sans KR'; color:#111; line-height:1.12; letter-spacing:-.02em; }
.card .body   { font:400 40px 'Noto Sans KR'; color:#111; line-height:1.5; margin-top:32px; }
.card .page   { font:600 24px 'JetBrains Mono'; color:#6B7280; }  /* 02 / 07 */
```

> 활용: 표지는 `.kicker`(주제 라벨) + `.head`(한 문장) + `.page`. 본문 카드는 `.head`(소제목) +
> `.body`(3줄 이내). 강조어 1개만 `color:#D97706`. 그 외 색은 쓰지 않는다.
